# Phishing Email Detection Model

A machine learning project that detects whether an email is **Phishing** or **Safe** using Scikit-learn.

## Features

- Train on phishing and safe emails
- TF-IDF text feature extraction
- URL feature extraction
- Random Forest Classifier
- Accuracy Score
- Classification Report
- Confusion Matrix
- Predict new emails

## Technologies

- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Structure

```
Phishing_Email_Detection/
│
├── dataset/
│   └── emails.csv
├── model.py
├── requirements.txt
├── README.md
```

